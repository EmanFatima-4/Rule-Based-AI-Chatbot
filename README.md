# Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot developed using Python. The chatbot interacts with users by recognizing predefined keywords and patterns and providing relevant responses.

## Features
- Interactive command-line interface
- Predefined responses for common questions
- Colorful chatbot messages using Colorama
- Help command to display available options
- Exit command to end the conversation
- Beginner-friendly implementation

## Technologies Used
- Python
- Colorama

## How to Run
1. Clone the repository:
   git clone https://github.com/EmanFatima-4/Rule-Based-AI-Chatbot/

2. Navigate to the project folder:
   cd rule-based-ai-chatbot

3. Install dependencies:
   pip install colorama

4. Run the chatbot:
   python chatbot.py

## Example Commands
- hello
- how are you
- what is your name
- help
- bye
